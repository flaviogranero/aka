# Aka

A CLI tool to automate Inaka Workflow tasks.
[https://github.com/inaka/inaka_corp/wiki/Inaka-Workflow](Inaka Workflow)

## Installation

Add this line to your application's Gemfile:

    gem 'aka'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aka
